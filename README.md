
# Customer Churn Prediction System (FUTURE_ML_02)

**Internship Task 2 – Machine Learning**

This repository contains a complete pipeline to build a Customer Churn Prediction System:
- data preprocessing
- model training (XGBoost)
- prediction/inference script
- example notebook and a minimal Streamlit app stub
- instructions and requirements

## Quick start

1. Create a virtual environment and install requirements:
```bash
pip install -r requirements.txt
```

2. Put your raw dataset (e.g., `churn_raw.csv`) in `data/raw/`.

3. Preprocess the data (generates `data/processed/churn_cleaned.csv`):
```bash
python src/preprocess.py --input data/raw/churn_raw.csv --output data/processed/churn_cleaned.csv
```

4. Train the model (saves model to `models/churn_model.pkl` and metrics to `results/`):
```bash
python src/train_model.py --data data/processed/churn_cleaned.csv --model_path models/churn_model.pkl --results results/
```

5. Predict using the trained model (outputs `results/predictions.csv`):
```bash
python src/predict.py --model models/churn_model.pkl --input data/processed/churn_cleaned.csv --output results/predictions.csv
```

6. (Optional) Run the Streamlit app stub:
```bash
streamlit run app/app.py
```
